AI Resume Analyzer
This project compares a resume with a job description and highlights matching and missing skills along with a match score.

What it does
- Compares resume and job description  
- Calculates a match score  
- Shows matched and missing skills  

How to run
pip install -r requirements.txt  
python ui.py  

Tech used-
Python  
NLTK  
Scikit-learn  
Gradio
